Scripts are executed every time a specific event occurs on a page. Scripts may be assigned to a number of elements through instrinsic event attributes. onload -> run a script when the broswer finishes loading a page. onclick -> run a script when the pointing device button is clicked over an element(picture, button, so something) onmouserover -> run a script when the pointing device is moved onto an element onfocus -> run a script when an element receives focus
